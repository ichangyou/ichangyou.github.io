读《男人来自火星，女人来自金星》

发表于 2019-08-28 更新于 2023-07-08 分类于 读书笔记 阅读次数：

关于作者

约翰·格雷，心理学博士，国际知名人际关系和情感问题研究专家。

关于本书

此书被誉为“关于两性情感关系最著名的作品”。作者在咨询调查了25000人后，用整整7年的时间，提出了一个形象有趣的“火星与金星理论”，点出了两性关系的本质：男人来自火星，女人来自金星，他们天生不同。

火星人的特质

火星的人生目标是创造改变，而金星的人生目标是开心快乐。

人生的意义和目标对女人来说是甜点，但却是男人的主食；同样，人生的爱和幸福对男人来说是甜点，但却是女人的主食。

火星人的特质。来自火星的男人习惯于单打独斗，坚持自力更生，不到万不得已的时候，绝不会向其他人寻求帮助。

火星人有个“洞穴机制”，在面临压力的时候，他会躲到远离喧嚣的洞穴中，独自思考自己当下面临的问题，专注地寻求解决方案，进行自我修复。如果找到了解决问题的方法了，他就会钻出洞穴，精神焕发，恢复成原来的模样。如果没有找到解决方案的话，他就会继续待在洞穴里苦思冥想，做一些小事来转移注意力，直到想出解决方案。这就是火星人的洞穴机制。

既然火星人的洞穴机制是无法避免的，那么当火星人钻进洞穴闭关修炼时，金星人应该怎么做呢？首先，金星人要明白，洞穴机制是火星人特有的减压机制，并不是存心要排斥和忽略她。接下来，她还可以适当找些消遣来打发时间，比如，读一本书、听听音乐、健身锻炼、写写日记等等。这样做不仅可以转移自己的注意力，还能减轻火星人的负担。等到火星人走出洞穴的时候，她还可以想出一些策略，提醒火星人关注她的需求。

金星人的特质

金星人，她们也有着独特的个性，那就是她们的语言表达系统。人们常说，女人心海底针。女孩的心思你别猜，也猜不明白。这是因为来自金星的女人说的语言跟火星人不同。在火星上的男人，话语是用来传递信息，表述事实的。而在金星上，语言更重要的作用是表达情感。女人是天生的修辞家，尤其擅长运用各种夸张、隐喻和归纳的手法，来抒发心中的所思所想，其实更多地是在抒发情绪。面对女人含蓄和戏剧化的语言表达时，男人习惯性地从字面上理解女人的话，很难听懂这些话背后的含义，从而做出错误的回应。

其实男人只要明白，女人说话是种情绪表达，不能从字面上理解，要适应她夸张的表达方式和戏剧性的语气。最最重要的是，要懂得倾听。这样，在日积月累的探索和实践下，男人就会了解女人的思维和感知方式，从而听懂她的言外之意、弦外之音了。

倾听是男人满足女人情感需求的首要法则，是进行有效沟通的前提。在用心倾听时，男人就会发现女人的抱怨并不是在责备自己，只是短暂的情绪表达。女人也会十分感激男人认同了她的内心情绪和感受，满足了她的情感需求。这样一来，不仅避免了争吵，还加深了彼此的爱和认可

抱怨、争吵、愤怒

两性关系面临的头号难题就是男人容易通过跟伴侣争吵以表达自己愤怒之情。但这样做无疑会导致两人在情感上缺乏安全感。女人一旦没了安全感，就会让自己的心坚硬起来，将心门紧锁，不再流露出任何信任和脆弱的情绪。由于无法产生足够的女性激素，她也就无法打开心扉去感受爱与激情。

女人只有在感受到自己可以安心地表露心里的情绪时，才会在个人的情感关系中逐渐放开真实的自己。男人如果能从男性的一面客观对待女人的情绪，而不是站在女性的一端，用愤怒的情绪来回应她，就能给女人一种安全感，让她觉得你不会因她说的某些话或做的某些事让你心烦而就此离开或伤害她。

如果一个女人不能分享自己的感受，那么她最后都会或多或少变得更加悲观，滋生出更多消极的想法和情绪。

过去，男人保护女人的人身安全；而现在，女人需要男人创造一种情感安全，让她们能做最真实的自己，表达真实的感受。

许多为妻子和家庭做了很多贡献的丈夫以为只要承担所有非日常性事务就够了，所以常常会在没有意识的情况下忽略了妻子的心情，没有给予她们需要的情感支持，如喜爱、赞美、沟通、关注与浪漫。这往往会造成大问题。

男女双方对抱怨的反应不一样。男人的抱怨会让女人觉得不堪重负，虽然她之后会因为他的抱怨而付出更多，但会觉得不堪重负，更会感到怨愤。而当女人向男人抱怨时，男人会觉得自己受到了别人的控制，最终会让他不再关心她是否快乐，不再愿意付出。

男人在有压力的时候，最不应该做的就是向妻子分享自己消极的情绪，哪怕不是在抱怨她。如果需要征询意见，那么他也应该要先做一些洞穴时间里的活动来提升一下体内的睾丸素水平，让自己冷静下来。可以找其他人或教练、心理咨询师聊，也可以做祷告或冥想，抑或做一些社会实践活动，如登山、打高尔夫、和男性朋友出去散散步等。做完了这些之后，如果还是感到心烦，那么他可以用写日记的方式来再次感受那些情绪。

男人不抱怨，不哭诉，他身边的女性伴侣就会感到安心和支持。

小结

男人和女人天生不同，在认知这个世界的时候会采用完全不同的方式，会产生差异、分歧。男人和女人不会因为相爱而消除彼此的不同，我们与其被彼此的差异困扰着直到对爱情与婚姻失去信心，不如积极地拥抱彼此的差异，重新认识对方，用恰当的方式进行高效沟通。

在爱中成长并不要求十全十美。假如我们已经很完美了，那么就不会有提升空间。只有面对自身的不完美，让自己做出改变，才能真正体会到无条件的爱。
