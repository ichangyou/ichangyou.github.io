读《见识：商业的本质和人生的智慧》

发表于 2019-08-17 更新于 2023-06-01 分类于 读书笔记 阅读次数：
一直很喜欢吴军的书，这是我读完他的第六本书（数学之美，大学之路，浪潮之巅，硅谷之谜，文明之光）了。以后他的其它书还会继续读下去。

一直很喜欢吴军的书，这是我读完他的第六本书（数学之美，大学之路，浪潮之巅，硅谷之谜，文明之光）了。以后他的其它书还会继续读下去。

《见识：商业的本质和人生的智慧》是根据吴军老师在 “得到” App 专栏的内容重新补充、调整后的作品。

不久前，吴军博士在一个访谈中聊了他对谷歌、苹果、微软、腾讯、百度等的看法，因为涉及到前东家，估计有人不开心了，或者根本就是木秀于林而已，网上很多人黑他，不少人说他的坏话。

吴军老师的涉猎之广让人惊叹，人文、技术、文学、音乐、绘画、摄影，书或专栏均有涉及，而且都不是浅尝辄止。

这样的人，一次不够注意的言论，也会引起群愤，可见无论谁，都不能取悦所有人。再厉害的人也不能让所有人满意，所以，走自己的路，让别人说去吧。我们只要能够获得自己最在意的人的认可就可以了。太在意外人的看法，会活得很累。

听了很多吴军老师的专栏，读了他不少的书，他对我最大影响就是让我有一个更好的学习和做事方法。他说的谷歌管理方法（OKR），我一直在使用。OKR，在工作中，在学习中（比如做读书规划），都是很不错的一种高效率做事方式。

还有他的理性的投资观，对我影响也很大。加上前辈的指点，现在我的观念完全被颠覆。如果不是他们对我影响，可能我还要经过多年才能开始现在的理财实践。所以说，人生中遇到贵人（前辈）指点，可以少走很多弯路。

我时常在想，如果很久以前，我就有高人指点，上学时候，我不会走那么多弯路，不会在自己的情绪中一直走不出来，如果以前处事方法得当，或许现在我会更厉害。但是过去的已经过去，往者不可谏，现在只希望来者犹可追。希望以后自己可以做得越来越好。

闲言少叙，书归正传，开始说说本书对我的影响。

谈谈见识

很多时候，成败与否取决于见识的高低，而不是自己简单的努力。今天，由于交通和通信技术的发展，我们增加见识要比过去容易得多。但在我们心中，有时依然有一道围墙，阻碍了我们的见识。我们提升境界的阻力很多时候在我们自身，而不完全在环境。

比贫穷更可怕的事情是缺乏见识、缺乏爱、缺乏规矩，是比缺钱更可怕的事情。没有钱，有一辈子的机会能够获得，而缺乏这三样东西，后天再获得的难度就非常大了。

理性的投资观

生活在物质世界里，就不能回避物质观，特别是金钱观。事业成功，挣到了钱，却不会花、不会理财，可能最后白忙活一场；没有挣到钱，空有一腔理想抱负，可能也不过是幻想。因此，金钱观不仅会对人的事业产生影响，也会决定人的幸福。

年轻时候，作为工薪阶层，一定要学会理财。可以根据自己的情况，把工资的百分之二十，百分之五十，甚至百分之八十，投资到股指基金、房产、股市以及其它合理的地方中去，而不是把所有钱放进银行。

每月把自己的一部分暂时用不到的钱投入到股指基金中，就相当于为以后存钱。以后会发生什么事情，谁也不敢保证。现在好，并不代表以后也会好。所以趁年轻，一定要多存粮，要未雨绸缪，这样以后真有事情了，才不会担心，才能轻松应对，也不会把自己捆绑在一个工作上一刻不敢停。

投资合理的话，多年后，还会因此赚到不菲的投资收入。而把钱放在银行，是永远跑不赢通货膨胀的。

把钱投入股指，或者房产，手上没有太多的现金流，或者你会担心，但是现金流不是资产，只有能为你赚钱的才是资产。股票、基金、房产，这些东西可以为你一直赚钱。

每个月都要还房贷，会觉得有压力？你就把它当作一种投资好了。有的人不还房贷，还不心安呢。加上通货膨胀的因素，你还的房贷，会随着时间的推移，越来越不算钱。十年前的一万块钱，和现在的一万块钱购买力一样么？那再过十年呢？

所以，可以趁早买房就趁早买房，当然买房最好在一线城市。

再有，年轻时，不能一心只想着怎样攒钱，更多的还是需要多想办法开辟赚钱渠道。你只有几万块钱，全部投资，也不会有很多的收益，只有想法大幅度提高自己的薪资，努力赚更多的钱，投资更多，才能赚取更多。工作的前几年，不要太在意自己的工资，要想办法提高自己的能力，尽快提高自己的薪资水平才是正路。

开始的时候，可以用自己的时间赚取一份收入，但是那不是长久之计。长久的方式是，你需要把自己的时间售卖 n 份，写书，写软件，制作在线教程等，把你的一份时间产出售卖 n 份儿， 那样你才能更快达到财富自由。

拒绝伪工作

提高工作效率是成功最关键的因素之 很多人试图在短时间里完成很多工作，这其实是办不到的，唯一能够控制的就是少做一点事情。效率高低不取决于开始了多少工作，而在于完成了多少。很多我们看上去非做不可的事情，其实想通了并没有那么重要，因为它们是伪工作。所以，无论是在职场上还是在生活中，提高效率都需要从拒绝伪工作开始。

OKR 是 Objectives Keys Results 的首字母缩写，即目标和衡量目标是否达成的关键结果。谷歌的每个员工每个季度之初都需要给自己定个或者几个目标，每个人的 OKR 大约半页纸长，写好后放到自己在公司里的网页上，这样大家都可以看到。如果谁没有制定 OKR 也目了然，即使没有人催促他，大家看到他的网页上面一片空白，他自己也会不好意思。这其实在管理上已经起到了最基本的监督作用。到了季度结束之前，谷歌的每一个人会给自已的目标完成情况打分。完成了，得分就是 1；如果部分完成，得分是 0 到 1 之间的一个数字。谷歌强调每一个人制定的目标要有挑战性，因此如果一个人完成目标的得分情况总是 1，并不能说明他工作好，而是目标定得太低。大部分情况下，大家完成目标的得分在 0.7~0.8。当然，一个卒季度开始时的想法，和后来完成的任条可能会有差异，早期没有想到的事情后来可能做了。因此，在总结季度工作时，可以增加当初没有制定的目标；对于不打算完成的目标，或者已经过时、不再有意义的目标，不能删除，但是可以说明为什么没有做完。

为自己制定合适的 OKR 季度计划，把年计划拆解为四个季度去完成，这样会比单独制定年度计划好很多。每个季度都去总结与反思，可以更好地去实现自己的年度计划。

好好说话

通过讲话交流想法水平的高低在很大程度上决定了一个人能否成功。

我因为沟通交流能力太低，在生活上吃了大亏，代价惨重。真是追悔莫及。如果以前就注意多学习一些沟通技巧，或许不会是现在这样情况。

幸福是目的，成功是手段。赚再多的钱，再成功，如果没有那个人一起分享，也不会感到幸福。

做人需要做减法

随着年岁的增大，随着自己的自由时间越来越少，我们有很多兴趣想去经营，有很多事情想去做，但是就是没有时间。一个人的精力是有限的，时间也是有限的，在有限的时间里，应该想办法多做成事，而不是多做事却事事都做不好。

吾生也有涯，而知也无涯，我们需要学会放弃一些东西，需要学会做减法。

有些事情是西瓜，有些事情是芝麻。我们需要学会多获得西瓜，而放弃一些芝麻。舍得，舍得，有舍才有得。

商业的本质是让人多花钱，而不是省钱。

商业的本质是让人多花钱，而不是省钱。至于如何让人们愿意花钱，这是艺术。我们可能更需要独立思考，从事物的本质出发，找到那些随大溜的人忽视掉的机会。

以后的商业会是共享经济、跟踪经济、合作经济、 众筹经济，机会会有很多，就看你有没有那个见识，就看你能不能抓住机会。

创业不是从 0 到 1，做到从 0 到 1，有了质的改变，但是想要最终成功，还需要很多路去走。以往的很多成功公司就是这样，走完了从 0 到 1，还要走很多路，才能最终达到伟大。

一个小公司，就像一只蚂蚁，已经存在的巨头公司就像大象，在巨头的压力下，蚂蚁的生存空间是很小的，蚂蚁如何战胜大象？这需要小公司有小公司的优势。小公司应该在某个领域做到精，做到完胜大公司，而不应该在很多领域都去尝试。小公司比钱，比人，都比不过大公司，那只能在精这方面做功夫。

希望未来，我不再是一个工薪阶层。
